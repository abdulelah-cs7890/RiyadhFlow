# RiyadhFlow 🚗

RiyadhFlow is a smart route planning web application designed to help users in Riyadh choose optimal travel routes and departure times based on traffic patterns.

## Features (Planned)
- Interactive map with route visualization
- Smart route suggestions
- Best time to leave recommendations
- Saved frequent trips

## Tech Stack
- Frontend: Next.js (React)
- Maps: Mapbox API
- Backend: (Planned)
